# リリースノート

## [0.0.1] - 初回リリース

### 新機能

- **選択テキストから Backlog にタスク作成**
  - ページ上で選択したテキストを、サイドパネルの「課題の詳細」に自動で反映
  - Chrome のサイドパネルでタスク作成フォームを表示
  - プロジェクト・課題種別・優先度・担当者・期限・詳細・添付ファイル対応
  - プロジェクト・担当者の検索（Select2 による絞り込み）
  - 最近使ったプロジェクト・担当者・メンションの記録

- **コンテキストメニュー**
  - テキスト選択後に右クリック → 「Backlog にタスク作成」でサイドパネルを開き、選択テキストを詳細欄に貼り付け

- **Backlog API 連携**
  - API キーと Backlog の URL（例: `https://your-space.backlog.com`）を設定画面で登録
  - プロジェクト一覧・参加ユーザー・課題種別・優先度を API から取得し、6 時間ごとにキャッシュを更新
  - サイドパネルから「今すぐ更新」で手動同期が可能
  - 課題の作成（概要・詳細・担当者・期限・添付・メンション）

- **オプション設定**（サイドパネル内「設定」から変更可能）
  - 送信後、作成した Backlog タスクの画面を開く / バックグラウンドで開く
  - ポップパネルで開く（Arc ブラウザ推奨）。拡張アイコン右クリックのコンテキストメニューからもオン/オフ可能
  - Backlog 連携: API キーの設定・変更、プロジェクト・担当者情報の手動更新
  - デベロッパーモード: タスク登録 API を送信せずリクエスト内容をコンソールに出力（デバッグ用）

- **表示・操作性**
  - システムのダークモード（`prefers-color-scheme: dark`）に応じたテーマ切り替え
  - アイコンクリックでポップアップとしても開けるオプション（設定で切り替え）

### 技術情報

- Chrome Extension Manifest V3
- 権限: `contextMenus`, `storage`, `activeTab`, `sidePanel`, `alarms`, `windows`, `scripting`
